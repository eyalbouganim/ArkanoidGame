# Arkanoid Game (Java)

An enhanced Arkanoid-style game built in Java, emphasizing clean Object-Oriented design and modular architecture.  
The project was developed as part of an academic OOP course and includes multiple levels, scoring, level progression, and a flexible code structure for future extensions.

---

## Features

- Four playable levels with increasing difficulty  
- Responsive paddle and ball mechanics  
- Score tracking and real-time on-screen updates  
- Pause and resume support (P to pause, SPACE to continue)  
- Game-over and win screens  
- Clear and modular class structure for adding new levels and mechanics  
- Observer pattern for decoupled and event-driven updates  
- Code designed with readability and maintainability in mind

## How to Run

This project requires **Java (JDK)** and the **JVM** installed.

1. Clone or download the project
2. Ensure JDK is installed and available from the command line
3. Navigate to the `src` directory
4. Run the following file as the program entry point: `Ass6Game.java`



Inside `Ass6Game.java`, you can choose to:

- Play the full game (all levels in order), or  
- Start from a specific level for testing or practice
